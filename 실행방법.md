# YJHWell 실행 방법 (오프라인 · Python)

2D 영구자석 모터 설계/해석 프로그램. **인터넷 없이** 로컬 Python으로 UI를 띄워
작업할 수 있습니다. (의존성 설치 때 한 번만 인터넷이 필요하고, 그 뒤로는 완전 오프라인.)

---

## 0. 준비물
- Windows + **Python 3.12** (3.13/3.14는 일부 과학 라이브러리 휠이 없을 수 있어 3.12 권장)
- 확인:
  ```
  py -3.12 --version
  ```
  안 깔려 있으면 https://www.python.org/downloads/ 에서 3.12 설치
  (설치 시 "Add python.exe to PATH" 체크).

---

## 1. 가장 쉬운 방법 (배치 파일, 더블클릭)
폴더 안에서:
1. **`setup.bat`** 더블클릭 — 가상환경(.venv) 생성 + 의존성 설치 (인터넷 1회 필요)
2. **`run.bat`** 더블클릭 — UI 실행 (이후 오프라인)

---

## 2. Python 명령으로 직접 실행 (수동)
폴더(예: `C:\Users\user\Desktop\YJHWELL`)에서 PowerShell/명령프롬프트를 열고:

```bat
:: (1) 가상환경 생성 — 최초 1회
py -3.12 -m venv .venv

:: (2) 의존성 설치 — 최초 1회 (인터넷 필요)
.venv\Scripts\python -m pip install --upgrade pip
.venv\Scripts\python -m pip install -r requirements.txt

:: (3) 실행 — 이후 언제나 오프라인 가능
.venv\Scripts\pythonw run.py      :: 창만 (콘솔 로그 없음)
.venv\Scripts\python  run.py      :: 콘솔 로그도 같이 보기
```

설치되는 패키지(고정 버전, `requirements.txt`): numpy, scipy, shapely, triangle,
pyqtgraph, pillow, PyQt6.

---

## 3. 프로그램 사용 흐름 (UI)
1. 리본 **Draw/Model** 탭에서 형상 작도 — 또는 메뉴로 **샘플 모터 삽입**(10극12슬롯)
   - **좌표계(Coordinates)**: `Relative CS`로 오프셋·회전된 작업좌표계를 만들면, 이후
     입력하는 좌표(좌표바·치수 다이얼로그)가 그 CS 기준으로 배치됩니다. `Set Global`로 복귀.
     (Maxwell의 Relative CS Offset/Rotate와 동일 — 캔버스에 빨강X/초록Y 트라이어드 표시)
2. 재질 할당 (Select Definition / View·Edit Material, B-H 곡선)
3. **Simulation** 탭 → Solve Setup → **Analyze** (B-field 오버레이)
   - **Nonlinear (BH)** 토글을 켜면 철심 포화를 반영한 비선형 해석
4. **Results** 탭:
   - **Torque Plot** — 무여자 코깅 토크 vs 회전각
   - **Back-EMF** — 무부하 역기전력 (3상 InducedVoltage A/B/C)
   - **Load Torque** — 정격 3상 전류 부하 토크
   - **Transient EMF** — 시간영역 역기전력 파형 (전기 2주기, vs 시간[ms])
   - **Core Loss** — 철손(Steinmetz/Bertotti: 히스테리시스/와전류/과잉) [W]
   - **Field Overlay** — 자속밀도 컬러 컨투어
   - **Param Sweep (Optimetrics)** — 설계변수 1개 스윕 → 토크/Bmax/Back-EMF/부하토크
     출력 표 + CSV 내보내기

> 극수·슬롯·속도·전류·turns·형상이 전부 변수라 **400W 한 종류가 아니라 다양한 모터**를
> 설계/해석할 수 있습니다.

## 4. 솔버 검증 (선택)
계산이 물리적으로 맞는지 자동 검증하는 하니스가 `feedback/`에 있습니다.
feedback-runner 스킬의 runner로 실행:
```
python <feedback-runner경로>\scripts\runner.py --config feedback\feedback.config.json
```
종료코드 0 = 모든 지표 허용오차 내(PASS). 설계무관 물리검증(3상 평형, e=-dλ/dt
자기일관성, dq vs FEM 토크, T∝I 선형성, 비선형 선형폴백 항등) + 회귀 베이스라인.

## 5. 문제 해결
- **`py`가 없다고 나옴** → `python -m venv .venv` 로 대체 (단, 기본 python이 3.12여야 함)
- **PyQt6 설치 실패 / 휠 없음** → Python이 3.12인지 확인 (`.venv\Scripts\python --version`)
- **`triangle` 빌드 오류** → 3.12 사용 시 미리 빌드된 휠이 설치됨. 3.13+에서 종종 실패.
- **창이 안 뜸** → `.venv\Scripts\python run.py` 로 실행해 콘솔 오류 메시지 확인
- **한글 경로/콘솔 깨짐** → 기능엔 영향 없음 (UI는 정상). 로그만 영향.
